# VLM/RAG統合 - Phase3 RAG統合実装 WBS (改訂版)

**プロジェクト:** VLM/RAG統合 - Phase3: RAG統合

**目的:** VLM処理によって生成されたMarkdownテキストを優先的に使用し、台帳データから検索用のチャンク（`ledger_chunks`）を生成する一連の処理を実装します。これにより、セマンティック検索の精度を向上させます。
