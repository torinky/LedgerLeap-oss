### 1.2 データ構造の理解

```php

    $table->boolean('optimized')->index();
    $table->string('mime', 500)->index();
    $table->text('path');
    $table->unsignedBigInteger('size')->nullable();
    // ...
    $table->timestamps();
    $table->softDeletes();

```

#### `AttachedFile` モデルの重要属性

```php
class AttachedFile extends Model
{
    protected $fillable = [
        'filename', 'hashedbasename', 'ledger_define_id',
        'ledger_id', 'column_id', 'mime', 'path', 'size', 'status',

    protected $casts = [
        'optimized' => 'boolean',
```

**重要な発見:**
- ⚠️ **`contain_content` はboolean型**で、テキストが抽出済みかどうかのフラグ

- ✅ **テキスト内容は `ledgers.content_attached` に保存されている**（Mroonga全文検索用）

## 2. 技術的制約と課題

### 2.1 制約事項

1. **テキスト抽出の動的生成が必要**
   - attached_files テーブルにはテキスト内容が保存されていない
   - 過去バージョンの添付ファイルを比較する場合、動的にTikaで抽出が必要
   - ⚠️ パフォーマンス懸念: 大きなPDFファイルの処理に時間がかかる

2. **バージョン管理の仕組み**
   - LedgerDiff に過去バージョンの `content_attached` が保存されているか不明
   - attached_files テーブルに過去バージョンの履歴が残るか不明
   - → **要調査**: バージョン間比較の実現可能性

3. **Mroonga全文検索の特性**
   - 現在のバージョンの `content_attached` は検索可能
   - 過去バージョンの content_attached は検索対象外の可能性が高い

### 2.2 現在の実装状況

#### ✅ 既に実装済みの機能

1. **SearchLedgersTool (Task 5.1完了)**
   - 添付ファイルのメタデータ取得（name, size, mime, column_id, hash）
   - Mroonga全文検索による添付ファイル内容の検索
   - format=summary でのユーザーフレンドリーな表示

2. **GetWorkflowHistoryTool**
   - ワークフローの履歴取得
   - バージョン情報の取得
   - ⚠️ 添付ファイルの履歴が含まれるか要確認

#### ❌ 未実装の機能

1. **抽出テキストの直接取得**
   - 特定の添付ファイルから抽出されたテキストを取得
   - 現在は `content_attached` 全体に含まれているが、個別取得不可

2. **過去バージョンの添付ファイル内容の取得**
   - LedgerDiff からの添付ファイル情報の取得
   - 動的なTika/OCR処理による過去ファイルのテキスト抽出

3. **添付ファイル間の差分比較**
   - 2つのバージョンの添付ファイル内容の自動比較
   - 変更箇所のハイライト

## 3. 実装オプションの評価

### Option 1: GetAttachedFileContentTool の実装（シンプル版）

#### スコープ
現在のバージョンの添付ファイルから抽出されたテキストを取得するツール。

#### 実装内容

```php
class GetAttachedFileContentTool extends Tool
{
    use AuthenticatedMcpTool;
    
    public function handle(Request $request): Response
    {
        $user = $this->authenticateOrError();
        if ($user instanceof Response) return $user;
        
        $ledgerId = $request->arguments('ledger_id');
        $fileHash = $request->arguments('file_hash');
        
        // 1. 台帳の取得と権限チェック
        $ledger = Ledger::find($ledgerId);
        if (!$ledger || !$this->canAccessLedger($user, $ledger)) {
            return $this->errorResponse('Permission denied or ledger not found');
        }
        
        // 2. content_attachedから該当ファイルの情報を取得
        $contentAttached = json_decode(json_encode($ledger->content_attached), true);
        $fileInfo = null;
        $columnId = null;
        
        foreach ($contentAttached as $colId => $files) {
            if (isset($files[$fileHash])) {
                $fileInfo = $files[$fileHash];
                $columnId = $colId;
                break;
            }
        }
        
        if (!$fileInfo) {
            return Response::json([
                'status' => 'not_found',
                'message' => 'File not found in this ledger'
            ]);
        }
        
        // 3. 抽出テキストの取得
        // content_attachedに含まれているテキスト内容を返す
        // ⚠️ 実際のデータ構造に依存 - 要調査
        
        return Response::json([
            'file_name' => $fileInfo['name'] ?? 'Unknown',
            'mime' => $fileInfo['mime'] ?? 'application/octet-stream',
            'size' => $fileInfo['size'] ?? 0,
            'column_id' => $columnId,
            'extracted_text' => $fileInfo['extracted_text'] ?? null,  // ← 要確認
            'extraction_status' => 'extracted',
        ]);
    }
}
```

#### 評価

**メリット:**
- ✅ 実装が比較的シンプル（2-3時間）
- ✅ 現在のバージョンの添付ファイル内容を明示的に取得可能
- ✅ 既存の権限チェック機構を活用

**デメリット:**
- ❌ `content_attached` にテキスト内容が含まれていない場合は機能しない
- ❌ 過去バージョンの添付ファイルには対応できない
- ❌ SearchLedgersTool + include_content との機能重複

**判定:** ⚠️ **保留** - content_attachedのデータ構造を確認してから判断

### Option 2: GetAttachedFileContentTool の実装（完全版）

#### スコープ
現在・過去バージョンの添付ファイルから動的にテキストを抽出するツール。

#### 実装内容

```php
class GetAttachedFileContentTool extends Tool
{
    use AuthenticatedMcpTool;
    
    protected TikaService $tikaService;
    
    public function handle(Request $request): Response
    {
        $user = $this->authenticateOrError();
        if ($user instanceof Response) return $user;
        
        $ledgerId = $request->arguments('ledger_id');
        $fileHash = $request->arguments('file_hash');
        $version = $request->arguments('version', null); // nullは最新版
        
        // 1. 権限チェック
        // ...
        
        // 2. AttachedFileレコードの取得
        $attachedFile = AttachedFile::where('ledger_id', $ledgerId)
            ->where('hashedbasename', $fileHash)
            ->first();
        
        if (!$attachedFile) {
            return $this->errorResponse('File not found');
        }
        
        // 3. テキスト抽出
        if ($attachedFile->contain_content) {
            // 既に抽出済み → content_attachedから取得
            $extractedText = $this->getExtractedTextFromContentAttached(
                $ledger, 
                $fileHash
            );
        } else {
            // 未抽出 → 動的にTikaで抽出
            $extractedText = $this->tikaService->extractText(
                storage_path($attachedFile->path)
            );
        }
        
        return Response::json([
            'file_name' => $attachedFile->filename,
            'mime' => $attachedFile->mime,
            'size' => $attachedFile->size,
            'extraction_status' => $attachedFile->status->value,
            'extracted_text' => $extractedText,
            'text_length' => mb_strlen($extractedText),
        ]);
    }
}
```

#### 評価

**メリット:**
- ✅ 未抽出のファイルにも対応可能
- ✅ 動的抽出により最新の内容を保証
- ✅ 過去バージョンへの対応可能性

**デメリット:**
- ❌ 実装が複雑（1-2日）
- ❌ パフォーマンス懸念（大きなファイルの処理）
- ❌ TikaServiceとの統合が必要
- ❌ 過去バージョンのファイルパスが取得できるか不明

**判定:** ⚠️ **要検討** - ユースケースの優先度次第

### 新しいヘルパーメソッド

```php
/**
 * Get extracted text preview from content_attached
 */
private function getExtractedTextPreview(
    array $contentAttached, 
    int $columnId, 
    string $hash, 
    int $length = 200
): ?string {
    if (!isset($contentAttached[$columnId][$hash])) {
        return null;
    }
    
    $fileInfo = $contentAttached[$columnId][$hash];
    
    // ⚠️ 実際のキー名はデータ確認後に修正
    $text = $fileInfo['extracted_text'] 
         ?? $fileInfo['text'] 
         ?? $fileInfo['content'] 
         ?? null;
    
    if (!$text) {
        return null;
    }
    
    return mb_substr($text, 0, $length) . (mb_strlen($text) > $length ? '...' : '');
}
```
