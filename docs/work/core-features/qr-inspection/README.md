## 採用した設計方針

### 基本コンセプト
- QRコード生成は外部サービスに任せる
- 台帳登録画面だけで完結
- 自動生成値（入力者名、日時）はQRコードに固定しない

### 主要機能
1. URLクエリパラメータによる初期値設定
2. 台帳登録画面で「登録用URLを確認」ボタン
3. user_name カラムタイプ（組織階層対応、追加モード対応）

### 実装期間
**2.5日間（20時間）**
