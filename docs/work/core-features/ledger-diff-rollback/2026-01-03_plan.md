# 台帳差分表示拡充・ロールバック計画（詳細計画用ドラフト）

**最終更新:** 2026-01-03

## 1. 目的・背景
- 台帳詳細画面（基本情報タブ）の利便性を履歴/差分画面に反映し、差分閲覧体験を底上げする。
- ワークフロー無効の台帳を対象に、過去バージョンへのロールバックを安全に提供する（ワークフロー有効版は後続フェーズ）。
- `ledger_diffs` を活用し、過去バージョンの確認・比較・復元を一連の導線で実現する。

**記載範囲**
- 差分表示強化・ロールバック（ワークフロー無効版）の要件整理と全体計画
- 詳細画面基本情報タブの機能を履歴/差分ビューへ移植する対象の洗い出し
- WBS（詳細計画管理のための骨子）

**記載しない内容**
- ワークフロー有効時のロールバック仕様（後続フェーズで定義）
- LLM検索/要約仕様（`docs/work/llm-integration/`）
- 添付ファイル処理フロー詳細（`docs/work/ui-ux/attachment/`）

## 2. 前提・制約（遵守）
- Laravel 12 / PHP 8.4 / Livewire / Alpine.js / Sanctum。
- テナント: Featureテストで `tenancy()->initialize()` 必須。
- AsColumnArrayJson: `data_get()`禁止。配列直接アクセス。
- Mroonga: 複合インデックス不可。全文検索は OR 結合。
- Livewire: 公開プロパティは単一配列で管理（状態一元化）。
- 履歴ソース: `ledgers` + `ledger_diffs`（status/ワークフロー含む）。
- 添付ファイル Phase1-5: 画像OCR後の`.pdf`キー、PDFは元キーを尊重。
- ロールバック適用範囲: "ワークフロー無効の台帳" に限定（権限・承認フローは無効化orスキップ）。

## 4. ユースケース（ターゲット範囲）
1. 基本情報タブ相当のUI/操作性で履歴差分を確認（フィールド/添付/ステータス）。
2. 任意2バージョン比較（最新↔直前、任意選択）。
3. ワークフロー無効の台帳で過去バージョンへロールバック（即時適用）。
4. ロールバック後にスコアリング再計算・全文検索インデックス更新・添付メタ整合チェックを非同期実行。

---

## 5. 機能要件（改訂版・ワークフロー無効限定）
- **差分ビュー**
  - フィールド差分（テキスト/数値/日付/選択/スコア表示）。
  - 添付差分: 追加/削除/更新（画像→PDFキー変換考慮）。
  - ステータス差分: `status` 等の履歴表示。
  - 任意2バージョン比較と差分ハイライト/フィルタ（変更ありのみ）。
  - 基本情報タブの表示レベル・グループ開閉・Auto Link・タグ表示を踏襲。
- **ロールバック（ワークフロー無効時のみ）**
  - バージョン単位で `ledgers` を復元、`ledger_diffs` にロールバックイベント追記。
  - 権限チェック（ポリシー + フォルダ権限 + ロール）。承認フローは無効化/スキップ。
  - ロールバック後に非同期ジョブ: スコアリング再計算、全文検索インデックス更新、添付メタ整合。
- **UI/UX**
  - 詳細画面基本情報タブの操作系・スタイルを履歴/差分に反映。
  - 差分凡例/フィルタ、ロールバック確認ダイアログ。
- **非機能/セキュリティ**
  - 操作ログ: 誰がいつどのバージョンにロールバックしたか。
  - テナント分離の保証。パフォーマンスは履歴件数多を考慮（遅延ロード/キャッシュ検討）。

---

## 6. データ・設計メモ（更新）
- 差分元: `ledger_diffs` の content / content_attached / status / metadata。
- 添付キー: 画像→`.pdf`、PDFは元キー。差分生成でキー変換を共通化。
- ロールバック手順（ワークフロー無効版）
  1. 対象 `ledger_diff` を取得し権限チェック。
  2. トランザクションで `ledgers` を上書き、`ledger_diffs` にイベント追記。
  3. キュー投入: スコアリング再計算、全文検索インデックス更新、添付メタ整合。
- 差分生成: サーバー側で標準化された diff DTO を生成し、Livewireで描画。

---

## 7. テスト方針（改訂）
- Feature: `DatabaseMigrations` + `tenancy()->initialize()`。
- 差分表示
  - 2バージョン間でフィールド/添付/ステータス差分が正しく出る。
  - 添付OCR/PDFキー差分が正しい。
  - 表示レベル/グループ開閉/Auto Link/タグ表示が履歴ビューでも踏襲される。
- ロールバック（ワークフロー無効）
  - 権限なしで拒否、権限ありで成功。
  - ロールバック後に `ledger_diffs` へイベント追記、インデックス更新ジョブ投入。
- パフォーマンス: 履歴件数多の応答を確認。
