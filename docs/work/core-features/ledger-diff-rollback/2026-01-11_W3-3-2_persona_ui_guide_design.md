# W3-3.2 ペルソナ別UI調整（クイックアクション・ガイド）詳細設計

**最終更新:** 2026-01-11
**対象:** LedgerLeap v12.0 / Cycle 3
**関連Issue:** #39
**関連要件:** [W2-1.3 Persona UX Requirements](./2026-01-04_W2-1-3_persona_ux_requirements.md)

## 1. 概要

本ドキュメントは、Cycle 3 で実装する「ペルソナ別UI調整（クイックアクションボタン、初回利用時ガイド）」の詳細設計書である。
実務担当者（非パワーユーザー）が複雑な任意比較機能を迷わず、かつ効率的に利用できるようにするための補助UIを提供し、既存の「基本情報タブ」での比較機能との相乗効果を高める。

### 1.1 背景と課題

- **基本情報タブの比較機能:** 現在、基本情報タブの「変更内容を表示する」トグルにより、履歴タブで選択されたバージョンとの比較が容易に行えるようになっている。
- **課題:** 基本情報タブで比較表示を行った際、比較対象（過去バージョン）の編集者情報が表示されない。また、ヘッダーに情報を出しすぎるとUIが煩雑になるが、承認ステータスは重要。さらに、編集者の詳細情報（ポップアップ等）も確認したい。
- **解決策:** ヘッダーに「バージョン」と「ステータス」を集約し、フッターに「両バージョンの編集者情報」と「ユーザー詳細ポップアップ (`user-card-popover`)」を配置する。

## 2. 基本情報タブでのユーザー体験 (UX) 向上

実務担当者は主に「基本情報タブ」を利用するため、履歴タブへ移動することなく必要な情報（誰が何を変えたか）にアクセスできる動線を構築する。

### 2.1 課題の再構成

- **現状:** 比較表示（変更内容を表示する）をオンにしても、過去バージョンの編集者が見えない。特定の過去バージョンと比較するには履歴タブに移動して選択する必要がある。
- **改善の方向性:**
  1. **情報の可視化:** 比較対象の編集者情報を常時表示する。
  2. **行動の誘発（ナッジ）:** 更新情報に「変更点を見る」リンクを添え、ワンクリックで比較表示モードに誘導する。
  3. **タブ内完結:** 比較対象のバージョンを基本情報タブから切り替えられるようにする（オプション検討）。

### 2.2 更新情報からのナッジ (Nudge)

- **場所1: 直前比較への誘導**
  - **位置:** 基本情報タブ下部の「最終更新者」エリア付近。
  - **文言:** `ledger.diff.nudge_view_changes`: 「直前の内容と比較する」
  - **動作:** クリックすると「変更内容を表示する」トグルが ON になり、直前バージョンとの比較が表示される。
- **場所2: 履歴一覧への誘導**
  - **位置:**
    1. 「直前の内容と比較する」リンクの横。
    2. 比較表示中のヘッダー。
  - **文言:** `ledger.diff.nudge_view_history`: 「すべての履歴・他のバージョンと比較」
  - **動作:** クリックすると「更新履歴」タブへ切り替え、ユーザーを詳細な履歴管理へと導く。

## 3. ヘルプガイド（ツールチップ）

### 3.1 構成

- **場所:** 基本情報タブの「変更内容を表示する」トグル付近。
- **アイコン:** `o-question-mark-circle`
- **表示形式:** ツールチップ。
- **文言:**
  - `ledger.workflow.guide.details_compare`: 「直前または『更新履歴』タブで選択した過去バージョンと内容を比較表示します。」

## 4. 情報の集約と配置の最適化

ステータス系情報はヘッダー、履歴詳細はフッターに整理する。

### 4.1 ヘッダーとフッターの役割分担

- **LedgerDiffViewer (ヘッダー):**
  - バージョン比較の方向性（Ver. 4 ← Ver. 3）を表示。
  - **承認ステータス:** ワークフロー有効時、現在の承認状態を表示。
  - 誘導リンク（他のバージョンと比較）。
- **基本情報タブ (フッター):**
  - 編集者詳細情報を表示（本バージョン・比較対象）。
  - **`user-card-popover` の導入:** 更新履歴タブと同様のユーザー詳細ポップアップを表示し、クリックやホバーで編集者の詳細を確認可能にする。
- **デザイン案 (フッター):**
  ```text
  [ 本バージョン (Ver.4) ] (popover: 編集者A) (2026/01/11)
  [ 比較対象 (Ver.3) ] (popover: 編集者B) (2026/01/10)
  ```

## 5. UIモックアップ（イメージ）

```text
[ 基本情報 ]
-----------------------------------
(カード内)
[ 変更内容を表示する ] (?) ← ヘルプ

(比較表示中: LedgerDiffViewer)
[ 比較中 ] Ver. 4 ← Ver. 3 [ 承認待ち ] [ 他のバージョンと比較 ]

...台帳内容...

-----------------------------------
(カード下部/フッター)
[ 本バージョン ] [x] 編集者A (2026/01/11 10:00) ← ポップアップ有効
[ 比較対象 ] [ ] 編集者B (2026/01/10 15:30) ← ポップアップ有効

[ 変更点を確認する ] [ すべての更新履歴を表示 ]
```

## 6. 実装方針

### 6.1 Livewire (Show)

- `targetDiffId` をリセットまたは直前バージョンにセットするナッジ用のメソッドを追加。

### 6.2 Blade (ledger-diff-viewer.blade.php)

- ヘッダーにバージョン番号と承認ステータスバッジ（ワークフロー有効時）を表示。

### 6.3 Blade (show.blade.php)

- フッターに現在の編集者と比較対象の編集者を並べて表示するロジックを追加。
- 編集者名部分に `x-ledger.user-card-popover` を適用。
- ナッジリンクとヘルプアイコンを追加。

- [ ] 無効化条件（最新時/Ver.1時）が正しく動作するか。
- [ ] ツールチップが適切な位置で表示されるか。
- [ ] 基本情報タブと比較表示が同期し、かつ両方の編集者情報（ポップアップ含む）が表示されるか。
