# コア機能 (Core Features)

このカテゴリには、LedgerLeapアプリケーションの根幹をなす主要な機能に関する実装計画や作業ログを格納しています。

## 📚 サブカテゴリ

- ### [ワークフロー (workflow)](./workflow/)
  承認フローなど、業務プロセスの中核をなす機能に関するドキュメント。

- ### [アクセス制御 (access-control)](./access-control/)
  権限管理やアクティビティログなど、セキュリティと監査の基盤に関するドキュメント。

- ### [添付ファイル (attachment)](./attachment/)
  ファイル添付、OCR、セキュアダウンロードなど、添付ファイルに関連する機能のドキュメント。

- ### [自動リンク (auto-link)](./auto-link/)
  テキスト内の特定文字列から関連情報へ自動的にハイパーリンクを生成する機能に関するドキュメント。
