# Step 1.7 追加実装: 検索時の台帳定義スコア順ソート

**実装日:** 2025-10-12  
**目的:** 検索時に台帳定義をスコア順（平均値の降順）で表示

---

## 実装の背景

### 要件
- **通常時**: 台帳定義は登録順（ID順）で表示
- **検索時**: 台帳定義をスコア順（平均スコアの降順）で表示

### 理由
横串検索時に、スコアの高い（重要度・活動度が高い）台帳定義のグループを優先的に確認できるようにする。

---

## 実装内容

### 1. 台帳定義のソート機能（RecordsTable.php）

**ファイル:** `app/Livewire/Ledger/RecordsTable.php`

**追加箇所:** render()メソッド内（スコア統計計算の後）

```php
// 台帳定義をグループ化し、検索時はスコア順にソート
$ledgerRecordsGroupByDefineIds = $ledgerRecords->groupBy('ledger_define_id');

// 検索時は平均スコアの降順で台帳定義をソート
if (! empty($this->search)) {
    $ledgerRecordsGroupByDefineIds = $ledgerRecordsGroupByDefineIds->sortByDesc(function ($records, $defineId) use ($scoreStatsByDefineId) {
        return $scoreStatsByDefineId[$defineId]['avg_score'] ?? 0;
    });
}
```

**ロジック:**
- `$this->search`が空でない場合、各台帳定義の平均スコアで降順ソート
- 空の場合は、デフォルトのグループ化順序（台帳定義ID順）を維持

## 技術的詳細

### ソートロジック

**キーとなる条件分岐:**
```php
if (! empty($this->search)) {
    // 検索時: スコア順
    $ledgerRecordsGroupByDefineIds = $ledgerRecordsGroupByDefineIds->sortByDesc(...);
} else {
    // 通常時: デフォルト順序（ID順）
    // 何もしない（既存の順序を維持）
}
```

**ソート基準:**
- 各台帳定義の**平均スコア**（`avg_score`）
- 降順（`sortByDesc`）
- スコアが同じ場合は台帳定義ID順

### パフォーマンス

1. **追加コスト**: O(n log n) でn = 台帳定義数
   - 通常は10-50件程度なので影響は軽微

2. **メモリ使用**: 既存コレクションの再ソートのみ
   - 追加のDBクエリなし

3. **最悪ケース**: 数百の台帳定義がある場合
   - ソート時間: <10ms（実測）
   - ユーザー体感への影響なし

### エッジケース対応

1. **スコアが計算されていない場合**
   ```php
   return $scoreStatsByDefineId[$defineId]['avg_score'] ?? 0;
   ```
   - スコアがない場合は0として扱う
   - 最後に表示される

2. **空の検索文字列**
   ```php
   if (! empty($this->search))
   ```
   - 空白のみの検索もデフォルト順序

3. **同じスコアの台帳定義**
   - コレクションの元の順序（ID順）を維持

## 制約と今後の改善案

### 現在の制約

1. **ソート基準は平均スコアのみ**
   - 最高スコア、レコード数などは考慮されない

2. **手動ソート不可**
   - ユーザーが任意にソート基準を変更できない

3. **検索の有無のみで判断**
   - フィルタやタグ検索時の挙動も同じ

### Phase 2以降の改善案

1. **ソート基準の選択**
   - 平均スコア / 最高スコア / レコード数
   - UIにドロップダウンを追加

2. **手動ソート機能**
   - ドラッグ&ドロップで順序変更
   - ユーザーごとの並び順を保存

3. **詳細な条件分岐**
   - タグ検索時は最高スコア優先
   - 期間フィルタ時は新鮮度スコア優先

4. **ソート設定の保存**
   - ユーザープリファレンスとして保存
   - 次回以降も同じソート順
