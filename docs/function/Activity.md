# 変更管理機能

## 概要

変更管理機能は、LedgerLeap において、モデルに対する変更を記録する機能です。この機能により、誰が、いつ、どのモデルをどのように変更したのかを把握できます。
`Spatie\Activitylog` を利用して、ユーザーの操作を記録します。

## 機能詳細

### 変更の記録

* モデルに対して行われた変更は、`App\Models\CustomActivity`に、変更履歴として記録されます。
* 変更される内容は、`Spatie\Activitylog` に設定されています。
* 変更されたモデルの`created`, `updated`, `deleted`が記録されます。
* 変更履歴は、台帳や、台帳定義に紐づきます。
* `App\Models\LedgerDiff`にも、変更内容を記録します。

### 変更履歴の確認

* 変更履歴は、詳細画面で確認できます。
* 変更前の値と変更後の値を比較できます。
* 誰が変更したか、いつ変更したか確認することができます。

### ユーザーのアクティビティ

* ユーザーの操作を記録します。
* `Spatie\Activitylog`で管理します。
* 変更履歴として、`App\Models\CustomActivity`に記録されます。
* いつ、だれが、どのように変更したか確認することができます。

## 関連ファイル

* `App\Models\CustomActivity`: 変更履歴を記録するモデル。
* `Spatie\Activitylog`: ユーザーのアクティビティを記録するパッケージ。
* `App\Models\LedgerDiff`: 台帳データの変更履歴を管理するモデル。
