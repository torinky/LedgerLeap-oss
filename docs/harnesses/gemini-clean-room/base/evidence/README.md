# Evidence directory

このディレクトリは contaminated run / clean-room run の比較証跡を置くための場所です。

## 推奨内容

- `evidence-template.md` をコピーした比較メモ
- `/memory show` の要約メモ
- `/skills list` の要約メモ
- スクリーンショット
- bootstrap discovery の応答差分

## 注意

- `workspace/` の下に置かず、discovery 対象外として扱うことを推奨
- runtime context に混ざるファイルは置かない
