# JpDatetimeService

## 目的

`JpDatetimeService` は、PHPの標準的な日付・時刻関数を拡張し、和暦（元号、和暦年）や日本語の曜日、午前/午後といった日本特有の表現に対応した日付フォーマット機能を提供します。これにより、アプリケーション全体で一貫した日本語の日時表示を実現します。

## クラス概要

*   **クラス名**: `App\Services\JpDatetimeService`
*   **役割**: 日本語の日時フォーマット機能を提供します。

## 主要な公開メソッド

*   **`date(string $format, ?int $timestamp = null): string`**:
    *   目的・機能: PHPの `date()` 関数と同様に、指定されたフォーマット文字列に基づいて日時を整形します。和暦（元号、和暦年）、日本語曜日、午前/午後といった日本特有のフォーマット記号をサポートします。
    *   引数:
        *   `$format`: 日時フォーマット文字列。`J` (元号), `b` (元号略称), `K` (和暦年 - 元年表記), `k` (和暦年), `x` (日本語曜日), `E` (午前/午後) などの独自記号が利用可能です。
        *   `$timestamp`: 整形するUNIXタイムスタンプ。`null` の場合は現在のタイムスタンプを使用します。
    *   戻り値: `string` - 整形された日時文字列。

## 依存する他のクラスや設定

*   なし（PHPの標準関数と内部の静的プロパティにのみ依存）

## その他

*   元号の定義はクラス内の `$gengoList` プロパティにハードコードされています。
*   日本語曜日の定義はクラス内の `$weekJp` プロパティにハードコードされています。
*   午前/午後の定義はクラス内の `$ampm` プロパティにハードコードされています。
