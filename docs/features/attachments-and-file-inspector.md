# 添付ファイルとファイルインスペクター

## 概要

LedgerLeap では、台帳にファイルを添付して管理できます。添付ファイルはテナントごとに分離して保存され、権限に基づいてアクセスを制御します。ファイルインスペクター、OCR 処理、VLM（Vision Language Model）抽出などの機能により、添付ファイルの内容を検索・確認できます。

## 添付ファイル管理の必要性

業務データには、台帳のカラムだけでは表現しきれない情報が含まれます。請求書の PDF スキャン、現場の写真、設計図面、外部システムから出力された CSV など、ファイルとして管理する必要がある情報を台帳にひもづけて保存し、必要なときに素早く参照できるようにします。

## 添付ファイルの管理方法

### ファイルの添付

台帳の作成時または編集時に、ファイルを添付できます。対応しているファイル形式は次のとおりです。

- **画像**: JPEG、PNG、GIF、WebP、BMP
- **ドキュメント**: PDF
- **表計算**: Excel（.xlsx、.xls）、CSV
- **テキスト**: プレーンテキスト（.txt）
- **その他**: システム設定で許可された形式

添付時には、ファイルの役割（証憑、参考資料、出力結果など）を指定できます。

### ファイルのダウンロード

添付ファイルは、台帳の詳細画面からダウンロードできます。ダウンロード URL はテナント単位でスコープされ、認証済みユーザーのみがアクセスできます。

### ファイルインスペクター

ファイルインスペクターは、添付ファイルの内容をブラウザ上で確認するための機能です。対応しているファイル形式については、ブラウザ上で直接プレビューを表示できます。

インスペクターでは次の操作が可能です。

- 画像ファイルの拡大表示
- PDF のページ送り
- テキストファイルの内容表示
- OCR テキストの表示（PDF 画像から抽出された文字列）
- VLM 抽出結果の表示（画像から自動抽出された説明文）

### OCR 処理

PDF ファイル（スキャン文書など）が添付されると、バックグラウンドジョブで OCR 処理が実行されます。抽出されたテキストは検索可能になり、キーワード検索の対象に含まれます。

OCR 処理の結果は、ファイルインスペクターの「OCR テキスト」表示で確認できます。また、OCR テキストを埋め込んだ PDF をダウンロードすることもできます。

### VLM 抽出

画像ファイルが添付されると、VLM（Vision Language Model）による自動抽出処理が実行されます。VLM は画像の内容を解析し、次のような情報を自動的に抽出します。

- 画像に写っているものの説明
- 文字情報（看板、書類などに記載されたテキスト）
- 数値やコード類の読み取り

抽出結果は台帳の検索対象となり、キーワード検索でヒットするようになります。

### 非同期処理

OCR 処理と VLM 抽出は、バックグラウンドジョブとして非同期に実行されます。ファイルを添付したあと、即座に結果が表示されるわけではなく、処理が完了するまで待つ必要があります。処理状況はファイルインスペクターで確認できます。

## 各機能の使い分け

| 機能 | いつ使うか |
|---|---|
| ファイル添付 | 台帳に関連するファイルを保存する |
| ファイルダウンロード | 添付ファイルをローカルに保存する |
| ファイルインスペクター | ブラウザ上でファイルの内容を確認する |
| OCR 処理 | スキャン PDF から文字を抽出して検索可能にする |
| VLM 抽出 | 画像の内容を自動的に解析して説明文を生成する |

## 注意点と制約

- アップロードできるファイルの最大サイズは、システム設定に依存します。
- OCR 処理および VLM 抽出は非同期で実行されるため、ファイル添付から結果が利用可能になるまで時間がかかる場合があります。
- 対応していないファイル形式は添付できません。
- 添付ファイルのダウンロードには、フォルダに対する適切な権限が必要です。
- OCR や VLM の抽出精度はファイルの品質に依存します。スキャン状態の悪い文書や、画像内の小さな文字は正しく抽出できない場合があります。
