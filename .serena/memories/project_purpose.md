# LedgerLeap プロジェクト目的

## プロジェクト概要
LedgerLeapは、組織内の情報管理と共有を効率化するための**Webベース台帳管理システム**です。

## 主な目的
- 散在しがちな業務記録やノウハウ、各種情報を一元的に管理
- 全文検索（添付ファイル含む）による必要な情報への迅速なアクセス
- 柔軟な権限管理による適切な情報共有の実現
- 詳細な活動履歴とアクセス追跡によるセキュリティと監査性の向上

## ターゲットユーザー
- 中小企業や大企業の部門・チーム単位（数人～数千人規模）
- 紙ベースや共有フォルダでの情報管理に限界を感じている組織
- 記録・検索・共有・監査を効率化したい組織

## 主要機能
- マルチテナント対応（複数組織・プロジェクトの論理分離）
- 柔軟な台帳定義（用途に合わせた自由な項目設定）
- 階層型フォルダ管理
- MySQL/Mroongaによる強力な日本語全文検索（類義語検索対応）
- Apache TikaによるOCR機能（画像・PDFからテキスト抽出）
- PaddleOCR-VLによる高精度VLM機能
- テナント横断の高度な権限管理
- アクセス権限の可視化と活動履歴追跡
- インテリジェントなワークフロー機能（多段階承認プロセス）
- 自動リンク機能（関連情報へのハイパーリンク自動生成）
- リアルタイム通知
- ユーザー中心のインターフェース（マイポータル、レスポンシブデザイン）
