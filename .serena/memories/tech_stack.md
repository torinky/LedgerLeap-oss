Backend: Laravel (PHP), Frontend: Livewire, Alpine.js, MaryUI (general users), Filament (admin), Database: MySQL/MariaDB + Mroonga (full-text search), File content extraction: Apache Tika, Development environment: Laravel Sail (Docker). Architecture: Standard web application (Web server, application server, DB server). Asynchronous processing: Redis and queue workers. API: Laravel Sanctum (SPA and API token authentication), RESTful API.
