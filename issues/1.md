### 概要
自動連番カラム（自動採番される ID/番号 等）の表示値が、一覧や詳細などのすべてのカラム表示で自動的にリンク化されることが期待されています。本 Issue では「実装がされているか」「テストが存在するか」を確認し、不足があれば対応方針を決めます。

### 背景／理由
- レコード間の参照を簡易にしたい。
- リンク化が抜けているとナビゲーション性が落ちる／ユーザビリティに影響が出るため。
- テストが無ければ将来のリファクタで壊れる可能性があるため、確認と補強が必要。

### 確認項目（チェックリスト）
- [ ] 実装: 自動連番カラムの値が一覧・詳細・エクスポートなど表示されるすべての箇所でリンク化されているか
- [ ] 単体テスト: この挙動を検証するユニットテスト／コンポーネントテストが存在するか
- [ ] 結合/UI テスト: 必要なら E2E や UI テストが存在するか（存在しない場合は追加検討）
- [ ] 既存 PR/コミット: 関連の PR やコミットがあれば参照リンクを貼る
- [ ] ドキュメント: 動作仕様が README/仕様書に記載されているか
- [ ] エッジケース確認: 空値、重複、特殊文字、権限による非表示などの挙動を確認

### 推奨対応（例）
1. コード検索で関連モジュール（テーブルレンダラ／カラムヘルパー／シリアライザ等）を特定する。
2. テストが無ければ単体テスト（レンダリング結果にリンクが含まれること）を追加する。
3. UI の場合はスクリーンスナップショットや E2E テストも検討。
4. 必要なら実装を補完する PR を作成する（PR にこの Issue を紐付ける）。
